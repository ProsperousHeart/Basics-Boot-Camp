# Strings

## About This Lesson

Master text manipulation with Python strings. Learn string operations, methods, formatting, and common string patterns.

### What You'll Learn
- String creation and syntax
- String indexing and slicing
- String methods (upper, lower, split, join, etc.)
- String formatting (f-strings, format(), %)
- String concatenation and repetition
- Common string operations
