# Tuples

## About This Lesson

Understand immutable sequences in Python. Learn when and why to use tuples instead of lists, and master tuple operations.

### What You'll Learn
- Creating tuples
- Tuple immutability
- Tuple packing and unpacking
- Tuple methods
- When to use tuples vs lists
- Named tuples
