# Operators

## About This Lesson

Master Python's operators for performing calculations, comparisons, and logical operations. This lesson covers arithmetic, comparison, logical, bitwise, assignment, and membership operators.

### What You'll Learn
- Arithmetic operators (+, -, *, /, %, //, **)
- Comparison operators (==, !=, <, >, <=, >=)
- Logical operators (and, or, not)
- Bitwise operators
- Assignment operators
- Membership operators (in, not in)
- Identity operators (is, is not)
