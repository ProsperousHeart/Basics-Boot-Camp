# Scope of Variables

## About This Lesson

Understand variable scope and namespaces in Python. Learn about local, global, and nonlocal scopes, and how Python resolves variable names.

### What You'll Learn
- Local vs global scope
- The LEGB rule (Local, Enclosing, Global, Built-in)
- The global keyword
- The nonlocal keyword
- Namespace concepts
- Variable lifetime
